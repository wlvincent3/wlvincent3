## Projects

### [Job Search Aggregator](https://github.com/wlvincent3/job-search-aggregator)
Multi-source job scraper that pulls from 6+ APIs and career boards (JSearch, USAJobs, Greenhouse, Lever, Workday, The Muse), scores each listing against a configurable profile, deduplicates across platforms, and syncs to Google Sheets. Cuts daily job-search overhead to a single terminal command.

### [CAPM Exam Prep Quiz](https://github.com/wlvincent3/capm-quiz-app)
Interactive React app covering all four CAPM exam domains — 76 scenario-based questions with full answer rationale and elimination logic. Built as a study tool, runs entirely in the browser.

### [Flight Search Tool](https://github.com/wlvincent3/flight-search)
Python script that searches Google Flights across multiple airport combinations, stop counts, and return dates in a single run, exporting a formatted Excel workbook for side-by-side comparison.
